# Week 2 – SOC Intermediate Operations

This section focuses on intermediate-level Security Operations Center (SOC) activities, bridging the gap between basic alert handling and advanced analysis.

## 📌 Topics Covered

* Alert Management and Classification
* Incident Response Documentation
* Alert Triage Techniques
* Evidence Preservation Basics
* SOC Workflow Simulation

## 📄 Contents

* Alert classification reports
* Incident response documentation
* Alert triage reports
* Evidence collection reports
* SOC investigation report

## 🛠 Tools & Concepts

* Google Sheets (Alert tracking)
* Basic Threat Analysis
* Incident Response Lifecycle
* SOC Workflow Practices

## 🎯 Objective

To build a strong foundation in SOC operations by practicing alert handling, triage, response, and documentation.

## 📈 Outcome

By completing this section, the learner gains practical understanding of real-world SOC processes including detection, analysis, and incident handling.

## 📌 Note

All activities are performed in a simulated environment for learning purposes.
