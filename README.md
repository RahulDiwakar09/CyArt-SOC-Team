# SOC Practical Tasks Repository

This repository contains hands-on Security Operations Center (SOC) practical tasks performed as part of cybersecurity training.

## 📂 Structure

* **Week-1:** Basic SOC Operations (Alerting, Incident Response, Triage)
* **Week-2:** SOC Intermediate Operations (Alert Management, Triage, Evidence Handling, Workflow Simulation)
* **Week-3:** Advanced SOC Analysis (Threat Intelligence, Log Analysis, Escalation)
* **Week-4:** Advanced SOC Operations (Threat Hunting, SOAR Automation, RCA, Adversary Emulation)

## 🎯 Objective

To develop practical SOC skills including detection, analysis, response, automation, and reporting.

## 🛠 Tools Used

* Wazuh
* VirusTotal
* AlienVault OTX
* TheHive
* Splunk SOAR (Conceptual)
* Velociraptor
* FTK Imager

## 📌 Note

All activities are simulated for learning and training purposes.
